# Windows Launcher

> 直角的美感

-----------

* 主题色
  * 浅色(白底，黑字）
  * 深色(黑底，白字)

* 背景
  * 无(遵从主题)
  * 图片
  * 高斯图片

* 主题色
  * 默认瓷砖背景
  * 文字方块背景
  * 高亮文本
  * 开关

* 瓷砖颜色
  * 默认主题色
  * 应用色
  * 专属色

* 瓷砖类型
  * 应用
  * 小部件
  * 文件夹
  * 特殊瓷块
  * 动态(多瓷砖叠加)

* 瓷砖大小
  * 1 x 1 小 无名称
  * 2 x 2 中 有名称
  * 2 x 4 长 有名称
  * 4 x 4 大 有名称

* 桌面大小
  * min 6
  * max 12

* 瓷砖间距
  * 小 5dp
  * 中 10dp
  * 大 18dp

* 图标包
  * 标准图标包
  * 专属图标包
